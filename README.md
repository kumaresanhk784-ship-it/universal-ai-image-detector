# Deepfake Image Detection System

## 📌 Description
This project detects AI-generated (deepfake) images using mathematical and statistical forensic analysis instead of deep learning.

## ⚙️ Features
- FFT (Frequency Analysis)
- Entropy Calculation
- Noise Variance Detection
- Color Correlation Analysis
- Blockiness Detection
- Confidence Score Output
- PDF Report Generation

## 🚀 Technologies Used
- Python
- Flask
- OpenCV
- NumPy
- Matplotlib

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000

## 📊 Output
- Real / Fake classification
- Feature values
- Confidence score
- FFT visualization
- PDF report

## 🎯 Objective
To detect deepfake images using explainable forensic techniques instead of black-box AI models.
